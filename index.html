🚀 Supabase Setup Required

To enable real-time cloud sync for orders and products, you need to connect a free Supabase project. Follow these steps:

Step 1 — Create Supabase Project Go to app.supabase.com → Sign up or login → Create a new project → Name it "sri-murugan-nursery"
Step 2 — Create Tables In your project → SQL Editor → Create the tables for products, orders, and users (schema provided in code comments)
Step 3 — Get API Keys Project Settings → API → Copy the URL and anon key
Step 4 — Paste Config Below Open this HTML file in a text editor and find the line:
const SUPABASE_CONFIG = { url: 'PASTE_YOUR_URL', key: 'PASTE_YOUR_KEY' }; Replace it with your copied config.
